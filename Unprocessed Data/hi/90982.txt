The HTML mark up produced by this template includes an hCard microformat, which makes the organisation's details parsable by computers, either acting automatically to catalogue article across Wikipedia, or via a browser tool operated by a reader, to add the subject to an address book. For more information about the use of microformats on Wikipedia, please see the microformat project.
